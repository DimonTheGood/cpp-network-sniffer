# C++ Network Sniffer (WSL2/Linux)

Простой сниффер сетевых пакетов на C++ с использованием библиотеки `libpcap`. 
Проект создан для изучения структуры сетевых протоколов и перехвата данных в локальной сети.

### Основные функции:
*   **IPv4/IPv6**: Анализ заголовков сетевого уровня.
*   **DNS Аналитика**: Декодирование имен сайтов (Query Name) из UDP/TCP трафика.
*   **DHCP Идентификация**: Перехват имен устройств (Option 12) в локальной сети.
*   **TCP/UDP**: Отслеживание портов отправителя и получателя.

### Как запустить:
1. Установите библиотеку: `sudo apt-get install libpcap-dev`
2. Скомпилируйте: `g++ main.cpp -lpcap -o sniffer`
3. Запустите: `sudo ./sniffer`
